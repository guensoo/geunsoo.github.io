# 🎨 포트폴리오 웹사이트

이 프로젝트는 제 **포트폴리오 사이트**입니다.  
**React + Framer Motion** 기반으로 제작되었습니다.

🔗 **[👉 사이트 바로가기](https://guensoo.github.io)**

## 💡 주요 특징
- 카드 기반 기술 스택 소개
- 인터랙티브한 호버 애니메이션
- 자기소개 및 프로젝트 섹션 구성
- 깔끔한 반응형 UI

## 🛠 사용 기술
- React
- Framer Motion
- JavaScript
- HTML/CSS
- GitHub Pages

## 📂 배포
- GitHub Pages를 통해 정적 웹사이트로 배포
